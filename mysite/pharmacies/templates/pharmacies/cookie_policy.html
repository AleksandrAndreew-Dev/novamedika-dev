Политика обработки файлов cookie

по состоянию на 11 июня 2025 г.

1. Общие положения

Издание Политики в отношении обработки файлов cookie является одной из обязательных мер по обеспечению защиты данных.

Для целей настоящей Политики под файлами cookie понимаются текстовые файлы, сохраняемые в интернет-браузере при посещении сайта для запоминания действий пользователя.

2. Категории обрабатываемых файлов cookie

На сайте обрабатываются только технические файлы cookie для:

  • Корректного функционирования сайта

Часть файлов cookie хранится в течение текущего сеанса и удаляется после закрытия браузера. Срок хранения остальных файлов cookie ограничен 1 годом с момента первого посещения сайта.

3. Правовые основания обработки файлов cookie

Технические файлы cookie используются для обеспечения функционирования сайта и обрабатываются независимо от согласия пользователя.

4. Трансграничная передача файлов cookie

Сайт не осуществляет трансграничную передачу файлов cookie.

5. Отключение файлов cookie

Пользователь может отключить использование файлов cookie в настройках браузера:

  • Mozilla Firefox
  • Google Chrome
  • Safari
  • Opera
  • Microsoft Edge

Примечание: отключение технических cookie может привести к некорректной работе сайта.
